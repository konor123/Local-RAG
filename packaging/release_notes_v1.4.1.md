## v1.4.1

### LLM 주도 검색 오케스트레이션
- 모든 통합 UI 질문이 검색 실행 전에 반드시 LLM Planner를 거치도록 변경했습니다.
- `file`, `content`, `hybrid`를 코드가 먼저 고르는 라우팅 모드가 아니라 LLM이 선택하는 도구로 정리했습니다.
- LLM이 요청한 도구 실행 순서를 그대로 보존하고, 기존 파일 검색 우선/내용 검색 생략 로직을 통합 흐름에서 제거했습니다.
- 검색 결과가 부족할 때 LLM이 재검색, 답변, 포기 여부를 다시 판단할 수 있는 제한된 검토 루프를 추가했습니다.

### 검색어 추출 개선
- 통합 검색, 하이브리드 검색, 수동 fallback 경로가 같은 검색어 추출 로직을 사용하도록 공통화했습니다.
- `최신`, `파일`, `찾아줘` 같은 보조 표현이 파일명 glob 패턴에 섞이지 않도록 개선했습니다.

### UI 정리
- 참고파일 카드와 코드블록의 다크블루 배경을 밝은 스타일로 변경했습니다.
- 참고파일 표시에서 파일명과 전체 경로 사이에 단독으로 나오던 `파일명` 줄을 제거했습니다.

### 검증
- 즉시 답변, 하이브리드 도구 선택, LLM 실행 순서 보존, 재검색 전략 수정에 대한 회귀 테스트를 추가했습니다.
